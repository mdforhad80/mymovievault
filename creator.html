🚀 Deploy to GitHub Pages

1. Upload all .html files to a GitHub repository. 2. Go to Settings → Pages → Source: Deploy from a branch → main → / (root). 3. Save – your site will be live at https://yourusername.github.io/repo-name

☁️ Deploy to Cloudflare Pages

1. Connect your GitHub repository to Cloudflare Pages. 2. Set build command: (leave empty), output directory: / (root). 3. Deploy – instantly live with a *.pages.dev domain.
